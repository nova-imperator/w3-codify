> and uses the brand look so all 5 match. **Text-free on purpose** — the app overlays the title.
> Ask for **16:9**. Save as `public/images/courses/<slug>.png`:

**Shared brand look (already baked into every prompt below):** dark near-black indigo
background `#0A0B14`, glowing indigo-violet `#6D5EF6` + electric cyan `#22D3EE` accents,
cinematic premium 3D render, soft volumetric glow, clean and modern, no text.
